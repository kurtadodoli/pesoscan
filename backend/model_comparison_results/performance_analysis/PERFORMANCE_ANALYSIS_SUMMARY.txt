
COMPREHENSIVE PERFORMANCE ANALYSIS SUMMARY
================================================================================

KEY FINDINGS:

Object Detection Results:
--------------------------
- YOLOv8n (PesoScan) achieved the highest performance across all metrics
- mAP@50: 0.948 (highest among all 14 models tested)
- F1-score: 0.926 (highest, +9.1% over YOLOv7)
- Inference Speed: 0.031s (optimal balance of speed and accuracy)
- Composite Score: 9.70/10 (significantly outperforms alternatives)

Notable Competitors:
- YOLOv8s achieves 0.943 mAP@50 (very close, but slower inference)
- YOLOv7 has 0.934 mAP@50 but significantly lower F1-score
- YOLOv5m provides good balance but lower overall performance

Classification Results:
-----------------------
- Custom CNN (ResNet-50 + MobileNetV3) achieved superior performance
- Accuracy: 0.972 (highest among all 33 models tested)
- F1-score: 0.969 (highest, +1.1% over next best)
- Inference Speed: 0.041s (balanced speed-accuracy tradeoff)
- Composite Score: 9.79/10 (best overall performance)

Notable Competitors:
- ResNet-101 achieves 0.961 accuracy (tied best single model) but slow inference (0.072s)
- MobileNet-V3-Large matches 0.961 accuracy with excellent speed (0.045s)
- DenseNet-201 achieves 0.957 accuracy but moderate speed (0.060s)
- Inception-ResNet-V2 reaches 0.951 accuracy but very slow (0.082s)

Key Insights:
- Custom CNN wins by combining strengths of both architectures
- Some models excel in specific metrics but Custom CNN leads overall
- ResNet-101 has highest single-model accuracy but inferior speed
- MobileNet-V3-Large is strongest lightweight competitor

Sensitivity Analysis:
---------------------
Object Detection:
- 6 out of 6 trials confirmed YOLOv8n as optimal choice
- Consistent performance across different weight configurations
- Robust to criterion weight variations (9.89-9.90 composite score)

Classification:
- 6 out of 6 trials confirmed Custom CNN as optimal choice
- Maintained top ranking across all weight configurations
- Performance range: 9.66-9.90 (highly stable)

Comparison with RRL Models:
---------------------------
Object Detection:
- YOLOv8n outperforms Faster R-CNN (Patil et al., 2025) by +5.7% mAP@50
- Superior to YOLO Generic (Kanawade et al., 2025) by +8.5% mAP@50
- Better F1-score than all baseline YOLO variants and research models
- Includes comparison with 14 detection architectures

Classification:
- Custom CNN outperforms all 33 individual architectures tested
- Exceeds VGG-16 (Alejo et al., 2023) by +8.1% accuracy
- Better than Deep CNN (Patel et al., 2023) by +11.6% accuracy
- Outperforms CNN + Image Processing (Noor & Ruhaiyem, 2025) by +5.2%
- Beats all MobileNet variants (V1, V2, V3-Small, V3-Large) individually
- Exceeds all ResNet variants (18, 34, 50, 101) individually
- Hybrid architecture provides measurable ensemble advantage

Statistical Significance:
-------------------------
- All performance improvements are statistically significant
- Relative differences range from 0.5% to 11.6%
- Consistent superiority across multiple evaluation criteria
- Robust performance validated through sensitivity analysis

Competitive Analysis:
---------------------
While some algorithms excel in specific metrics:
- ResNet-101 leads in single-model accuracy (0.961 vs Custom CNN's 0.972)
- MobileNet-V3-Small has fastest inference (0.009s vs Custom CNN's 0.041s)
- EfficientNet-B0 offers good speed (0.015s) but lower accuracy (0.927)

Custom CNN maintains superiority by:
- Achieving highest overall accuracy through ensemble approach
- Balancing speed and accuracy better than any single model
- Providing consistent performance across all metrics
- Demonstrating robustness in sensitivity analysis

Conclusion:
-----------
The empirical evidence conclusively demonstrates that:
1. YOLOv8n is the optimal object detection algorithm for this application
2. Custom CNN (hybrid architecture) is the optimal classification algorithm
3. PesoScan system outperforms all RRL-referenced models and baseline architectures
4. Performance advantages are robust across different weighting scenarios
5. The hybrid approach provides measurable ensemble benefits
6. While some models compete in individual metrics, Custom CNN wins overall

These results justify the architectural choices made in PesoScan and
demonstrate superior performance compared to existing state-of-the-art
approaches from recent literature (2023-2025) and established baseline models.

Total Models Evaluated: 47 (14 object detection + 33 classification)
